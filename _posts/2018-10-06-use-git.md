## 配置姓名和邮箱

首先，设置你的姓名和邮箱。这两个参数必须设置，否则无法进行后续add、commit和push等操作。
  
```js
git config --global user.name=a605605
git config --global user.email=413707989@qq.com
```

## Git初始化：

```js
git init
```
